# Cities Scraper.

### Cities Scraper is a script that get cities.csv file which contains table of country cities information.



## run script
```sh

virtualenv -p python3 venv

source venv/bin/activate

python main.py

Enter Country Name: Egypt

check 'data/cities.csv'

```

#### NOTE
##### Accuracy
	- EGYPT - 100%
	- whole world - 80%
